# 数学

---

```javascript
Math.round() // 四舍五入
Math.pow(x, y) //  x 的 y 次幂
Math.sqrt(x) // 开根号
Math.abs(x) // 绝对值
Math.ceil(x) // 向上取整
Math.floor(x) // 向下取整
Math.sin(x) 
Math.cos(x)
Math.log()
Math.exp()
Math.min() 
Math.max()
Math.random()

Math.E          // 返回欧拉指数（Euler's number）
Math.PI         // 返回圆周率（PI）
Math.SQRT2      // 返回 2 的平方根
Math.SQRT1_2    // 返回 1/2 的平方根
Math.LN2        // 返回 2 的自然对数
Math.LN10       // 返回 10 的自然对数
Math.LOG2E      // 返回以 2 为底的 e 的对数（约等于 1.414）
Math.LOG10E     // 返回以 10 为底的 e 的对数（约等于 0.434）
```
